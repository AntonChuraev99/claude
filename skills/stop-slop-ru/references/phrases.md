# Фразы под нож

Конкретные русские маркеры нейросетевого текста. Сгруппированы по типу. Принцип один: убрать обёртку, оставить факт.

## Дидактические вводные (разгон перед мыслью)

Нейросеть предваряет мысль объявлением о её важности. Вырежи объявление, скажи мысль.

- «Важно отметить, что…»
- «Важно понимать, что…»
- «Стоит отметить, что…»
- «Следует отметить / учитывать, что…»
- «Необходимо подчеркнуть, что…»
- «Нельзя не упомянуть…»
- «Не секрет, что…»
- «Хотелось бы отметить…»
- «Сразу оговоримся…»
- «Забегая вперёд, скажем…»
- «Как известно…»

## Шаблонные зачины

Пустой запев вместо начала по делу.

- «В современном мире…»
- «В современном динамично меняющемся мире…»
- «В эпоху цифровизации / цифровых технологий…»
- «В последнее время всё большую популярность приобретает…»
- «С каждым годом всё больше…»
- «Трудно переоценить значение…»
- «Данная тема отличается повышенной актуальностью…»
- «Сегодня как никогда…»
- «На протяжении многих лет…»

## Клише значимости

Абстрактная «роль» и «значение» вместо конкретного действия. Назови, что предмет делает.

- «играет важную / ключевую / значительную / решающую роль»
- «имеет большое / огромное значение»
- «является неотъемлемой частью»
- «занимает особое / центральное место»
- «служит / выступает напоминанием»
- «подчёркивает / отражает важность»
- «символизирует…»
- «это краеугольный камень…»
- «ключевой / поворотный момент»
- «нельзя переоценить»

## Слова-пустышки и маркетинговые усилители

Удали или докажи конкретикой. Эпитет без факта — пустой.

- «уникальный», «инновационный», «передовой», «революционный»
- «эффективный», «качественный», «оптимальный», «комплексный»
- «по-настоящему», «поистине», «действительно»
- «широкий спектр / ряд», «целый ряд», «множество»
- «богатый, яркий, разнообразный»
- «потрясающий», «впечатляющий», «захватывающий»
- «мощный инструмент», «надёжное решение», «гибкий подход»

## Неопределённость (страховка нейросети)

Модель уходит от прямого утверждения. Утверждай или режь.

- «может стать…», «может оказаться полезным», «могут повлиять»
- «в ряде случаев», «как правило», «зачастую», «в некоторой степени»
- «по мнению экспертов», «многие специалисты считают»
- «тот или иной», «определённую роль», «своего рода»
- «не исключено, что…»

## Концовки-ритуалы

Дай вывод по ходу текста. Отдельный финальный реверанс не нужен.

- «Подводя итог…»
- «В заключение хотелось бы отметить…»
- «Таким образом, можно сделать вывод, что…»
- «В целом…», «Вкратце…»
- «Надеюсь, эта статья была полезной»
- «Не забывайте, что…»
- «Помните: …»

## Мета-комментарии (текст про сам себя)

Текст должен двигаться, а не пересказывать свою структуру.

- «Давайте разберёмся…»
- «В этой статье мы рассмотрим…»
- «Как мы увидим далее…», «Как было сказано выше…»
- «Об этом — ниже»
- «Перейдём к следующему пункту»
- «Прежде чем перейти к…, давайте…»

## Канцеляризмы → живое слово

| Канцелярит | По-человечески |
|------------|----------------|
| осуществлять | делать |
| осуществление / реализация / обеспечение (отглагольные) | глагол: сделать, запустить, дать |
| является (X является Y) | X — это Y; или прямой глагол |
| обладает (X обладает Y) | у X есть Y |
| в целях | чтобы |
| в связи с тем что | из-за того что, потому что |
| в случае если | если |
| на сегодняшний день | сейчас, сегодня |
| в рамках | в, при, во время |
| данный, вышеуказанный, вышеупомянутый | этот, названный |
| в количестве N штук | N штук |
| по причине | из-за |

## Расщеплённые сказуемые (глагол-пустышка + сущ. → один глагол)

- «оказывать влияние» → «влиять»
- «принимать решение» → «решать»
- «вести борьбу» → «бороться»
- «производить осмотр» → «осматривать»
- «давать оценку» → «оценивать»
- «осуществлять контроль» → «контролировать»
- «вносить вклад» → ... (часто пустышка, переписать)
- «проводить анализ» → «анализировать»
- «принимать участие» → «участвовать»
