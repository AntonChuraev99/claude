# Stop Slop (русский)

Скилл, который вычищает признаки нейросетевого текста из русской прозы.

Русская адаптация [stop-slop](https://github.com/hardikpandya/stop-slop) Хардика Пандьи. Английский оригинал бьёт по английским тэллам (`-ly` наречия, em-dash, Wh-зачины, passive voice). В русском вредят другие приёмы, поэтому это не перевод, а отдельные правила.

## Зачем

Нейросеть пишет предсказуемо: канцелярит, дидактические вводные, штампы значимости, отглагольные существительные, монотонный ритм. Скилл учит Claude (и любую LLM) ловить эти приёмы по-русски и убирать их.

## Установка

**Claude Code:** положи папку в `~/.claude/skills/` (глобально) или в `.claude/skills/` проекта.

**Claude Projects:** загрузи `SKILL.md` и файлы из `references/` в знания проекта.

**Свои инструкции:** скопируй правила из `SKILL.md` в системный промпт.

**API:** добавь `SKILL.md` в системный промпт. Файлы `references/` подгружаются по необходимости.

## Что ловит

**Фразы** — дидактические вводные («важно отметить»), шаблонные зачины («в современном мире»), клише значимости («играет ключевую роль»), канцелярит, расщеплённые сказуемые, концовки-ритуалы. См. `references/phrases.md`.

**Структуры** — ложные контрасты («не просто X, а Y»), правило трёх, отглагольные существительные в начале предложения, пассив и ложная агентность, кривые деепричастные обороты, монотонный ритм. См. `references/structures.md`.

**Типографика** — шаблонное длинное тире, заголовки с заглавной у каждого слова, лишний жирный шрифт, ошибки с «однако» и «благодаря».

## Оценка

Поставь 1–10 по каждому измерению:

| Измерение | Вопрос |
|-----------|--------|
| Прямота | Утверждаешь или объявляешь? |
| Ритм | Живой или метроном? |
| Доверие | Уважает ум читателя? |
| Живость | Звучит как человек? |
| Плотность | Есть что вырезать? |

Ниже 35/50 — переписать.
