Положи текст в системный буфер обмена Windows через PowerShell `Set-Clipboard` — аналог хоткея «c to copy» в Claude Code, только как команда.

**Что копировать:**
- Если в `$ARGUMENTS` есть текст → копируй ровно его.
- Если `$ARGUMENTS` пусто → копируй последний значимый вывод из текущего разговора (URL, блок кода, команду, путь — то, что пользователь вероятнее всего хочет забрать). Если кандидатов несколько и неясно какой → спроси одной строкой, не копируй наугад.

**Как копировать надёжно** (произвольный текст: кавычки, `$`, `&`, многострочный) — single-quoted here-string не интерполирует `$` и бэктики:

```powershell
Set-Clipboard -Value @'
<текст сюда дословно>
'@
```

Закрывающий `'@` — на отдельной строке с нулевым отступом (иначе PowerShell не распознает конец here-string). Для однострочного текста без кавычек допустимо `Set-Clipboard -Value '<текст>'`.

**После копирования** подтверди одной строкой: короткое описание + длина в символах. НЕ печатай весь текст повторно. Пример: `✅ Скопировано в буфер: URL авторизации (312 символов)`.

Не используй `| clip` — он портит кодировку кириллицы/Unicode и добавляет лишний перевод строки. Только `Set-Clipboard`.
